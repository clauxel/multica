# Multica SaaS Site

This is a standalone static marketing site that does not depend on the original `multica` repository code.

## Pages

- `index.html`: homepage
- `scenarios.html`: scenario library
- `definition.html`: definition page
- `pricing.html`: pricing page
- `blog.html`: blog index
- `signup.html`: trial entry page
- `blog-why-cloud-agent-workspace.html`
- `blog-from-prompts-to-task-flow.html`
- `blog-solo-dev-agent-routines.html`

## Preview

You can open `index.html` directly in a browser.

If you want to preview it through a local server:

```bash
cd site
python -m http.server 8080
```

Then visit:

`http://localhost:8080`
